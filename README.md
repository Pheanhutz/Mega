# Mega

# Features: 
✔️ Auto Referral
✔️ Auto Task Execution
✔️ Auto Ping
✔️ Multi-threading & Multi-account Support

# 🖥 Instructions:

## Git Clone
```
git clone https://github.com/Pheanhutz/Mega.git
cd Mege
```

## Install dependencies:
```
npm install
```
## Edit ref_code
```
nano config.js
```
## Add Proxies
```bash
nano proxy.txt
```
## Run the Auto-Ref Script
```bash
node autoref
```
## After Using Auto-Ref (Run the Main Bot 
```bash
node main
```
